https://github.com/sephirothwzc/code-generator-wzc

使用说明：

```node
$ npm install code-generator-wzc --save-dev

$ yarn add code-generator-wzc -D
```

package.json

```md
"scripts": {
"codeGeneratorWzc": "node ./node_modules/code-generator-wzc/index.js"
}
```

run

```node
# run
$ yarn codeGeneratorWzc
```
